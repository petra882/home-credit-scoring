# Home Credit Default Risk — EDA и ML-модель кредитного скоринга

Разведочный анализ данных (EDA) и построение модели машинного обучения для предсказания невозврата кредита на датасете [Home Credit Default Risk](https://www.kaggle.com/c/home-credit-default-risk).

## Описание проекта

Проект выполняет задачу, типичную для риск-аналитика в банке: на основе исторических данных о заёмщиках выявить группы с повышенным риском невозврата и построить модель для автоматического pre-screening заявок.

## Проверенные гипотезы (EDA)

### 1. Влияние имущества на возврат кредита

Проверена гипотеза: *«Наличие машины или недвижимости снижает риск невозврата»*.

**Результат:** Гипотеза **не подтвердилась**. Доля невозвратов во всех четырёх группах (нет ничего, только квартира, только машина, всё есть) колеблется около 8% — значимых отклонений не обнаружено.

![Доли невернувших по наличию недвижимости](plots/Доли%20невернувших%20по%20наличию%20недвижимости.png)

### 2. Влияние возраста на возврат кредита

Проверена гипотеза: *«Молодые заёмщики менее надёжны»*.

**Результат:** Гипотеза **подтвердилась**. Обнаружена монотонная зависимость: чем старше заёмщик, тем ниже доля невозвратов.

| Возрастная группа | Доля невозвратов |
| :--- | :--- |
| 18-25 | 11.7% |
| 25-30 | 11.2% |
| 30-45 | 8.8% |
| 45-50 | 7.3% |
| 50+ | 5.6% |

![Доля невернувших по возрастам](plots/доля%20невернувших%20по%20возрастам.png)

### 3. Влияние типа занятости на возврат кредита

Проверена гипотеза: *«Тип занятости влияет на риск невозврата»*.

**Результат:** Гипотеза признана **нерепрезентативной**. Категории `Student`, `Businessman`, `Unemployed`, `Maternity leave` содержат слишком мало наблюдений (от 5 до 22), чтобы делать статистически значимые выводы.

![Доля невозвратов по типу занятости](plots/Доля%20невозвратов%20по%20типу%20занятости.png)

### 4. Кросс-анализ: тип кредита × возраст

Проверена гипотеза: *«Revolving loans безопаснее Cash loans для всех возрастных групп»*.

**Результат:** Гипотеза **подтвердилась**. Revolving loans **систематически безопаснее** Cash loans во всех возрастных группах. Разница достигает **двух раз** для групп 25-30 и 30-45.

| Возраст | Cash loans | Revolving loans | Разница |
| :--- | :--- | :--- | :--- |
| 18-25 | 12.5% | 9.3% | 3.2 п.п. |
| 25-30 | 11.8% | 6.6% | 5.2 п.п. |
| 30-45 | 9.2% | 5.2% | 4.0 п.п. |
| 45-50 | 7.5% | 4.9% | 2.6 п.п. |
| 50+ | 5.8% | 3.6% | 2.2 п.п. |

![Зависимость невозвратов от типа кредита и возраста](plots/Зависимость%20невозвратов%20от%20типа%20кредита%20и%20возраста.png)

### 5. Медианный анализ: почему Revolving loans безопаснее?

Проверена гипотеза: *«Revolving loans безопаснее, потому что их берут более обеспечённые люди»*. Сравнены медианные доход, сумма кредита и credit-to-income ratio по типам кредита.

**Результат:** Гипотеза **частично подтвердилась**. Доход заёмщиков почти одинаков (153 000 vs 135 000), но **сумма кредита у Cash loans в 2 раза выше** (540 000 vs 270 000). Как следствие, **credit-to-income ratio у Cash loans в 1.75 раза выше** (3.5 vs 2.0).

**Вывод:** Разница в уровне риска объясняется не типом занятости и не доходом, а **кредитной политикой банка**: по Cash loans выдаются значительно большие суммы относительно дохода заёмщика.

| Метрика | Cash loans | Revolving loans |
| :--- | :--- | :--- |
| Медианный доход | 153 000 | 135 000 |
| Медианная сумма кредита | **540 000** | **270 000** |
| Медианный credit-to-income | **3.5** | **2.0** |

## ML-моделирование

### Модель

**CatBoostClassifier** выбран как лучшая модель для задачи с сильным дисбалансом классов (92% / 8%).

**Параметры:**
- `iterations=200`, `learning_rate=0.1`
- `scale_pos_weight=11.5` — штраф за пропуск невозвратника

**Признаки:**
- Числовые: `age_years`, `credit_to_income`, `EXT_SOURCE_1/2/3`
- Категориальные: `NAME_CONTRACT_TYPE`, `NAME_INCOME_TYPE`, `CODE_GENDER` (OneHotEncoder)

### Результаты

| Метрика | Значение |
| :--- | :--- |
| **ROC-AUC** | **0.744** |
| **Recall (класс 1 — невозврат)** | **0.62** |
| **Precision (класс 1)** | 0.16 |
| **Accuracy** | 0.73 |

### Интерпретация

Модель ловит **62% будущих невозвратников** ценой того, что 84% предсказанных «невозвратников» на самом деле вернули бы кредит. Для кредитного скоринга это приемлемый компромисс: цена пропуска невозвратника (потеря тела кредита) выше, чем цена ложной тревоги (дополнительная ручная проверка заявки).

**Бизнес-сценарий:** Модель используется как первый этап pre-screening: 73% надёжных заёмщиков получают кредит автоматически, 27% направляются на ручную проверку. Нагрузка на отдел верификации снижается почти в 4 раза.

Важность признаков:

Топ-3 признака: `EXT_SOURCE_3`, `EXT_SOURCE_2`, `EXT_SOURCE_1` — внешние скоринговые баллы. Самостоятельно созданный признак `credit_to_income` вошёл в топ-4, подтвердив гипотезу EDA.

## API и Docker

Модель обернута в REST API (FastAPI) и упакована в Docker-контейнер.

**Эндпоинт:** `POST /predict`

**Пример запроса:**
```json
{
  "age_years": 55,
  "credit_to_income": 1.5,
  "EXT_SOURCE_1": 0.75,
  "EXT_SOURCE_2": 0.80,
  "EXT_SOURCE_3": 0.78,
  "NAME_CONTRACT_TYPE": "Revolving loans",
  "NAME_INCOME_TYPE": "Pensioner",
  "CODE_GENDER": "F"
}
```
